# Railway AI Command Centre Frontend

React, TypeScript, Tailwind CSS v3, Framer Motion, and Axios interface for the Railway AI Track Inspector backend.

## Run

```bash
npm install
npm run dev
```

The app expects the FastAPI backend at `http://localhost:8000`.

## Build

```bash
npm run build
```

## Pages

- Login
- Operations Dashboard
- Track Inspection Upload and Grad-CAM Viewer
- Prediction History
- Reports
- Model Information
- Settings
